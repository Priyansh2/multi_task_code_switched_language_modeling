Janiye kaun hain aapke liye best.\n\n\n\n\n0\n\n\n\n\nTech-News,\n\nTech-Tips\n\n\n\n\n\nA+\nA-\n\n\nPrint\nEmail\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\nJab aap kisi file, photo aadi ko store karne ke liye jaate hain toh sabse pahle yeh decide karna padta hain ki aap kaun saa cloud use kar rahe hain. Halaki sabhi cloud storage ka kaam ek hi hota hain, lekin yeh janna jaroori ho jata hain ki aapke data ko kaun sabse safe rakh sakta hain. to aaiye jane OneDrive, Dropbox aur Google Drive ke baare mein. \n\n\nCloud ka kya hain kaam \n\n\n\nKoi user jab apni file ya data ko store karta hain, to uska sidha makshad hota hain ki future mein iss data ko kabhi bhi restore kiya jaa sakta hain. Cloud mein kisi bhi data ko store karne ke kai faayde bhi hote hain. Isme sabse bada advantage yeh hain ki aap apni file ko kabhi bhi aur kanhi bhi access kar sakte hain. Iske saath hi yeh aapki file ko Backup ke roop mein bhi rakhta hain. Vaise ek nazar mein dekha jaye to sabhi cloud storage ka kaam ek jaisa hi hota hain, Lekin yahan yeh bhi dekhna padega ki kiski service asaan hain aur kiski mushkil.\n\n\n\n(1) One Drive \n\n\n\nSabse pahle baat karte hain One Drive ki. Yeh Microsoft ka Storage option hain. Yeh specially Windows users ke liye hain. Jinke pass Window 8 aur 8.1 version hain vah ise asaani ke saath access kar sakte hain. Yeh inke operating system mein pahle se mauzood hota hain. Halaki yeh Android aur ios mein bhi asaani se run kar sakta hain. Iske liye aapko Officially website par jakar OneDrive Android download karna hoga. Philhaal iss cloud storage mein aap kisi bhi tarah ki file store kar sakte hain. jisme photos, videos aur documents shamil hain. Ek baar aap isme koi file store kar dete hain to aap ise kanhi bhi doosre PC ya phone me access kar sakte hain.\n\n\n \n\nKya hain khasiyat :- OneDrive Service khas taur par window smartphone users ke liye banayi gayi hain. Jiske chalte yeh window operating system mein mauzood hota hain. Iske alawa yeh Ms-Office se kafi had tak juda hua hain. Yani ki jo log office purpose ke liye presentation ya photos ka work karte hain, unke liye yeh faaydemand hain. \n\n\nKahan rah gyi hain kami :- agar aap window device nahi use karte hain to aapko thodi preshani ho sakti hain. OneDrive use karne ke liye aapko Microsoft mein account sign up karna padta hain. Jiske chalte aap Outlook, Xbox Live aur anya Microsoft ki services ko access kar sakte hain. Halaki Android aur ios mein aisi koi suvidha nahi milegi. Iske alawa isme file save karne ki limit 20000 tak hain.\n\n\n\n(2) DropBox\n\nDropBox  sabse jyada popular cloud storage mana jata hain. Yeh kafi reliable hain, asaani se use kiya ja sakta hain aur set up friendly hain. Yeh ek aisa cloud hain jo kisi bhi OS mein access kiya ja sakta hain. Yani ki aap agar isme kisi file ko store kar dete hain, to ise dropbox website se, desktop applications for Mac, Windows aur Linux (Ubuntu, Debian, Fedora or Computer your own), or the iOS, Android aur Blackberry kisi bhi service ke jariye restore kiya ja sakta hain. Iske saath hi Dropbox mein store kiya hua data aap apne PC mein asaani se move kar sakte hain. Vahi yeh service Automatic hain , Yeh turant hi aapki file ko sink kar deta hain.\n\nDropbox ki Khasiyat :- Dropbox ki sabse badi khasiyat yeh hain ki, isme file store ki koi limit nahi hain. Iski strength kafi majboot hain. Iske saath hi yeh PC, Laptop aur smartphones sabhi mein ek jaisa hi work karta hain. Iski service kafi asaan hain aur ise koi bhi asaani ke saath use kar sakta hain.\n\nDropbox ki kamiyan :- Dropbox ke design ki baat kare toh user ko bahut hi kam pasand aati hain. Dropbox ka design iss cloud storage ki sabse badi kami hain. Halaki ki yeh asaan aur saaf- suthari hain, Lekin agar aapse kanhi bhi choonk ho gyi to file ka corrupt hona tay hain.\n\n(3) Google Drive\n\nOnline Office tool ka sabse best option hain, Google Drive. Yeh ek complete office suit hain, jisme cloud storage ki suvidha mauzood hain. Isme aapko word processor, spreadsheet application aur presentation builder bhi milega. Iske saath hi 15 GB ka free storage space bhi milega. Agar aapke pass gmail account hain toh aap ise asaani ke saath access kar sakte hain. Isme aap photos, Documents, videos sabhi ko store kar sakte hain. Vaise yeh Google ki service hain, aapko chromebox ki suvidha bhi milegi.\n\nKya hain khasiyat:- Agar aap Google Drive ki service ko use karna chahte hain toh iske liye aapke pass Google account ka hona Compulsory hain. Aap Gmail use karte hain to aapki Email se koi bhi file kuch hi click mein Google Drive mein Store ho jati hain. Iske saath hi agar aapka smartphone Google Account se chal raha hain to camera se click ki gayi sabhi photos automatically Google Drive mein Store ho jayengi.\n\nKahan rah gyi kami:- Jaisa ki bataya gya hain ki agar aapka Data Google Account se sink hain , tabhi yeh badhiya tarike se work karega. Yani ki sirf Google Account se sink hone par hi photos Automatic save ho payenge. \n\n\n\n\n\n\n\n\nPosted by\n\n\nHinglishpedia\n\n\n\n\n\n\n\n\n\n\n\n\nShare to:\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\nNextNewer Post\nPreviousOlder Post\n\n\n\n\n\n\n\nPost a Comment\n\n\n\naprieztmkrdezign\n218168578325095\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\nSubscribe to:\nPost Comments (Atom)\n\n\n\n\n\n\n\n\nLoading...\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\nPopulars\nComments\nArchive\n\n\n\n\nFeature Post\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\nPopular\n\n\n\n\n\n\n\n\n\nRukhe baalo ko Silky banane ke gharelu upay aur nuskhe (Tips for Dry Hair to make Silky)\nKya aap Rukhe baalo (Dry Hair) se preshan hain?
