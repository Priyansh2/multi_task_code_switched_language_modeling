Whatsapp download karte hi Verification message aane se pahle hi apna phone Flight Mode par kar dijiye. Aise mein Whatsapp dusra option dega. Isme "Verify through message" ka option choose kijiye aur apni E-mail id daliye. Send par click kar dijiye aur Sending message ko bhi cancel kar dijiye. Is tarah aapko Whatsapp par apna Mobile number nahi dalna padega. \n\n\n\n\n\nPDF aur Heavy File share karne ke liye :- Agar aap chahte hain ki Whatsapp ke jariye hi PDF aur heavy files ko share kar sake to iske liye aapko Whats Tools download karna hoga. Iske jariye aap 1 GB tak ki file share kar sakte hain. \n\n\n\nKhas person ke online hone ka pata :- Agar aap chahte hain ki kisi special person ke online aate hi aapko message mil jaye, to yeh mumkin hain. WhatsApp online history naam ka App download kijiye aur kamaal dekhiye. \n\n\n\nWhatsapp Message schedule :- Agar aap Whatsapp ke ke Messages schedule karna chahte hain, to Message Scheduler for WhatsApp download kijiye aur kamaal dekhiye.\n\nNaya sim par nahi hain jaroorat dubara whatsapp install karne ki ;- Agar aapne phone mein naya sim dala hain to aapko whatsapp dubara se install karne ki jaroorat nahi hain. Settings > Account > Change number on your WhatsApp par jayiye. Aur apna naya aur purana number feed kar dijiye. Iske baad done par click kijiye. Aapki puri chat vaapis aa jayegi.\n\nWhatsapp Last seen na badle ;- Agar aap chahte hain ki aapke Whatsapp ka Last Seen na badle aur aap kisi ka Message bhi padh le, to asaan tarika hain. Net off kar dijiye, Whatsapp kholiye aur dost ka message padh lijiye. Usay reply bhi kar dijiye. Ab dobara Net ON kijiye aur kamaal dekhiye. Aapka message to chala jayega lekin aapka  Last Seen nahi badlega.\n\nJaniye agar kisi ne kiya hain aapko block ;- Agar aapko shak hain ki kisi contact ne aapko block kar diya hain, to usay ek group mein Add kijiye. Agar usne aapko Block kiya hoga to yeh nahi ho payega aur aapko Yeh message dikhaayi dega "Not Authorised to Add this Contact".\n\nDelete kiye gye message dubara prapt karna :- Agar aap chahte hain ki aapko Delete kiye gye message bhi mil jaye to Phone ke Database Folder mein jayiye. Iska Path yeh hain. SD Card > WhatsApp > Data Bases. Yaha par aapko \'msgstore-2014-01-04.1.db.crypt aur msgstore.db.crypt naam ki kuch files milegi. Dusri File ko backup-msgstore.db.crypt naam se rename kijiye. Ab message wali file ko change kijiye aur File ka naam msgstore.db.crypt kar dijiye. Iske baad, Settings>Applications>Manage Applications>WhatApp par jayiye. Yaha Clear Data ke option par click kijiye. Saara Data delete ho jayega. Ab Whatsapp kholiye aur Data ko restore kar lijiye. Delete kiye gye sabhi message vaapis mil jayenge.\n\nJaniye aapke dost ne message kab padha  :- Yeh jyadatar logo ke saath hota kisi special one ko message karne ke baad hame intezaar rahta hain ki unhone message padha kab. Iska asaan tarika hain. Jo message aapne send kiya hain, usay select kijiye. Aapko upar delete button se pahle ek naya Symbol dikhaai dega. Yeh Info button hota hain.
